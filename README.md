# biostat721_project2
## Overview
This project analyzes monthly air pollutant trends (CO, PM2.5, and O3) in Durham County, NC from 2018–2020.  
The project includes:

- Cleaning and preparing EPA air quality measurements  
- Aggregating daily values into monthly summaries  
- Creating visualizations to illustrate pollutant patterns  
- Producing a reproducible Word report using R Markdown  

This analysis provides environmental context relevant to public health and sickle cell disease–related research.
